## 简介

本项目为自用Blog系统的移动管理端，主要初衷在于实现简单版的系统管理，方便管理员维护博客站点;
目前仍在技术选型：[Maui](https://github.com/dotnet/maui) or [Avalonia](https://github.com/AvaloniaUI/Avalonia)
